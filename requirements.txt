# StudyFlow Requirements

## System Requirements
- Node.js >= 16.0.0
- MongoDB >= 4.4
- Git

## Dependencies (Auto-installed via npm)
- express
- mongoose
- bcryptjs
- jsonwebtoken
- socket.io
- cors
- helmet
- compression
- express-rate-limit
- dotenv

## Frontend Dependencies
- Vue.js 3
- Tailwind CSS
- Phosphor Icons
- Chart.js

## Optional
- YouTube API Key (for music integration)
- MongoDB Atlas account (for cloud database)

## Environment Variables (.env file)
- MONGO_URI=your_mongodb_connection_string
- JWT_SECRET=your_jwt_secret
- FRONTEND_URL=your_frontend_url (optional)
- YOUTUBE_API_KEY=your_youtube_api_key (optional)
